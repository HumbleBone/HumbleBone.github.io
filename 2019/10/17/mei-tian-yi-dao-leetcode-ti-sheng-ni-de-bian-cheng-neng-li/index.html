每天一道leetcode提升你的编程能力

Question : 为什么突然会萌生出每天坚持编写一道 leetcode 来提升自己的编程能力呢？

Answer ：其实步入研究生已经一个多月了，这学期修了一门数据结构与算法设计，老师布置过几次大作业，感觉一道挺简单的题目也要花好多时间才能解决😂。再想想要是以后面试，就这种水平…….🤮 诶……这都是本科欠下的账😭，没办法，编程这种东西就是要每天坚持练，才能提升。同时，也是为了毕业之后，面试算法的时候不至于一道题目想半天，未雨绸缪吧。所以，就萌生这个想法🐡！

前面balabala了一大堆，我们马上进入正题，这篇文章主要记录如何高效使用 leetcode 编程，提升自己的代码水平。遇到这种问题，我们一般的做法肯定都是先知乎一手。不错，知乎上面有很多前辈给出了自己的经验 传送门👈。

选择什么语言

知乎上大部分人都是推荐使用 java 、 python 、 C++。不怎么推荐使用C语言，好像会遇到很多语言细节的问题，时间成本太高。同时我们主要学习的是遇到一个问题的想法和思路，所以这里我们选择 python 作为我们编程的学习语言。

有哪些开源的攻略

我主要了解到在 github 上有两个高星的项目，都是中文的。项目链接和简介我直接放在下面：

1、项目一

  • 项目地址： https://github.com/azl397985856/leetcode
  • 亮点：
    • LeetCode 经典题目的解析，包括思路、关键点和具体的代码实现。
    • 支持 java 、 python 语言，而且任然在更新中
    • 开放每天一道leetcode题目，供大家讨论

2、项目二

  • 项目地址： https://github.com/MisterBooo/LeetCodeAnimation
  • 亮点：
    • 将 LeetCode 上所有的题目都用动画的形式演示出来

上面两个项目我们主要用来学习思路，做完一遍之后能够实现自己的见解。同时在 LeetCode 官网上面也有很多优质的解答，都可以汲取别人的思路。

选择什么工具编写代码

当然，你可以直接在 LeetCode 官网上提供的在线编译器编写代码，这里我给大家推荐一种更加方便舒服的工具：VSCode + LeetCode 插件，可支持在 VSCode 中编译调试，并提交。

LeetCode 有两个 Endpoint，我们一般选择中国节点，该插件目前虽然只是预览版，但是我个人用下来基本没有什么问题。和网站上一样，可以根据难易、标签来选择题目。每次做完的题目都可以保存下来（需要自己创建一个文件夹），方便日后查看。

在使用过程中，我们选择一个题目之后，界面只会显示这道题目的简介。这时候，我们需要右键点击这道题目，选择 show the problem， 首次选择会让你选择编程语言和创建工作目录，之后就可以开始编程了。

当我们编写完一道题目之后，我们可以选择 Submmit|Test，进行测试或者提交。

如何最大化编程效益

因为是每天一道 LeetCode 编程题，所以我就要考虑在有效时间内如何最大化编程效益🧐？

1、按照 Tag 标签先由易到难专攻一个知识点的题目，一开始可能会比较吃力，后面应该就会好很多。

2、身边刚好有一本《算法导论》，可以结合书上的知识点，先做相关知识的阅读，之后针对这个知识点找相应的 Tag 刷题。

3、 每道题目做完之后都要记录自己的思路，答案不是最重要的。记录思路，留在日后总结。

4、将自己的思路上传到 Github 上。

最后放上我项目的地址： https://github.com/HumbleBone/LeetCode
