最近训练模型需要用到一些图片集，因此在网上搜索了相关的爬虫代码，最后总结了如何使用python爬取google和baidu的图片。就个人而言，百度图片的关键词爬取到的图片存在图片质量不是很高，分类不是很准确但是速度比较快。而google爬取到的图片质量相对而言比较高且分类准确，缺点是速度会比较慢。接下来开始介绍如何爬取这两个网站的图片：

python爬取Baidu图片

# coding:utf-8
import re
import requests
from urllib import error
from bs4 import BeautifulSoup
import os

num = 0
numPicture = 0
file = ''
List = []


def Find(url):
    global List
    print('正在检测图片总数，请稍等.....')
    t = 0
    i = 1
    s = 0
    while t < 1000:
        Url = url + str(t)
        try:
            Result = requests.get(Url, timeout=7)
        except BaseException:
            t = t + 60
            continue
        else:
            result = Result.text
            pic_url = re.findall('"objURL":"(.*?)",', result, re.S)  # 先利用正则表达式找到图片url
            s += len(pic_url)
            if len(pic_url) == 0:
                break
            else:
                List.append(pic_url)
                t = t + 60
    return s


def recommend(url):
    Re = []
    try:
        html = requests.get(url)
    except error.HTTPError as e:
        return
    else:
        html.encoding = 'utf-8'
        bsObj = BeautifulSoup(html.text, 'html.parser')
        div = bsObj.find('div', id='topRS')
        if div is not None:
            listA = div.findAll('a')
            for i in listA:
                if i is not None:
                    Re.append(i.get_text())
        return Re


def dowmloadPicture(html, keyword):
    global num
    # t =0
    pic_url = re.findall('"objURL":"(.*?)",', html, re.S)  # 先利用正则表达式找到图片url
    print('找到关键词:' + keyword + '的图片，即将开始下载图片...')
    for each in pic_url:
        print('正在下载第' + str(num + 1) + '张图片，图片地址:' + str(each))
        try:
            if each is not None:
                pic = requests.get(each, timeout=7)
            else:
                continue
        except BaseException:
            print('错误，当前图片无法下载')
            continue
        else:
            string = file + r'\\' + keyword + '_' + str(num) + '.jpg'
            fp = open(string, 'wb')
            fp.write(pic.content)
            fp.close()
            num += 1
        if num >= numPicture:
            return


if __name__ == '__main__':  # 主函数入口
    word = input("请输入搜索关键词(可以是人名，地名等): ")
    # add = 'http://image.baidu.com/search/flip?tn=baiduimage&ie=utf-8&word=%E5%BC%A0%E5%A4%A9%E7%88%B1&pn=120'
    url = 'http://image.baidu.com/search/flip?tn=baiduimage&ie=utf-8&word=' + word + '&pn='
    tot = Find(url)
    Recommend = recommend(url)  # 记录相关推荐
    print('经过检测%s类图片共有%d张' % (word, tot))
    # numPicture = int(raw_input('请输入想要下载的图片数量 '))
    numPicture = 500
    # file = raw_input('请建立一个存储图片的文件夹，输入文件夹名称即可')
    file = 'F:/VScodeProjects/test_python_environments/nba_images/landmark_1'
    y = os.path.exists(file)
    if y == 1:
        print('该文件已存在，请重新输入')
        file = input('请建立一个存储图片的文件夹，)输入文件夹名称即可')
        os.mkdir(file)
    else:
        os.mkdir(file)
    t = 0
    tmp = url
    while t < numPicture:
        try:
            url = tmp + str(t)
            result = requests.get(url, timeout=10)
            print(url)
        except error.HTTPError as e:
            print('网络错误，请调整网络后重试')
            t = t + 60
        else:
            dowmloadPicture(result.text, word)
            t = t + 60

    print('当前搜索结束，感谢使用')
    print('猜你喜欢')
    for re in Recommend:
         print(re, end=" ")

python 爬取Google图片

在网上搜索爬取google图片的时候，发现很多程序最多只能下载100张图片，或者就是没有很好的解决反爬虫机制。幸运的是，在我准备放弃的时候，意外发现github上竟然有开源了一个 google-images-download 的项目。项目有4540的star，并且附有详细的教程，个人亲测效果不错。

项目地址

前提

  • 自备好梯子，安装好google浏览器
  • 支持windows、mac、linux环境
  • 默认脚本不需要安装其他任何包，但是只能最多下载100张图片，如果想要下载100张以上的图片，需要另外安装Selenium包和chromedriver

下载

提供以下几种方法安装

1、using pip

$ pip install google_images_download

2、using CLI

$ git clone https://github.com/hardikvasa/google-images-download.git
$ cd google-images-download && sudo python setup.py install

安装chromedirver

安装地址

根据系统选择自己安装的版本

使用方法

1、使用终端：

$ googleimagesdownload [Arguments...]

具体使用如下：

googleimagesdownload -k "麦加大清真寺" -l 500 -o F:\VScodeProjects\test_python_environments\nba_images\landmak_3 --chromedriver=F:\VScodeProjects\google-images-download\chromedriver.exe
  • -k 关键词
  • -l 下载图片数量
  • -o 图片存储位置
  • –chromedriver chromedriver安装位置

2、编写python

from google_images_download import google_images_download

response = google_images_download.googleimagesdownload()
absolute_image_paths = response.download({<Arguments...>})

举个栗子

from google_images_download import google_images_download

arguments = {
    "keywords": "库里",

print(absolute_image_paths)
